Professional website for Full Aware - Driving AI Innovation.

## Services

1. **AI Memory Systems** - Extending AI capabilities with RAG architectures, vector databases, and intelligent document processing.

2. **Data Intelligence** - Clean, relevant, consistent data classification and organization for better AI decisions.

4. **AI Integration** - Integrating AI capabilities into workflows to plan for future outcomes.

## Features

- Light/Dark theme toggle (persisted in localStorage)
- Mobile responsive design
- Custom brain circuit logo branding
- Scroll animations on service cards
